# 🚀 كودي - تعلم البرمجة للأطفال

برنامج تعليمي تفاعلي لتعليم البرمجة للأطفال من سن ٦ إلى ١٠ سنوات باللغة العربية.

## 📚 محتوى البرنامج

- **الدرس ١:** ما هي البرمجة؟ - أساسيات وتعريف البرمجة
- **الدرس ٢:** المتغيرات - كيف نحفظ المعلومات
- **الدرس ٣:** الشروط - اتخاذ القرارات في البرمجة
- **الدرس ٤:** الحلقات - تكرار الأوامر بذكاء
- **الدرس ٥:** الدوال - إنشاء أوامر مخصصة
- **🎮 ملعب الكود:** كتابة وتشغيل كود بايثون مباشرة
- **🏆 الإنجازات:** نظام نجوم وأوسمة لتحفيز الأطفال

## 🛠️ طريقة التشغيل المحلي

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🌐 نشر على Streamlit Cloud

1. ارفع الملفات على GitHub repository
2. اذهب على: https://share.streamlit.io
3. سجّل دخول بـ GitHub
4. اضغط "New app"
5. اختار الـ repository والـ branch
6. في "Main file path" اكتب: `app.py`
7. اضغط "Deploy!" 🚀
